# 🛍️ ShopSphere – Modern Electronics E-Commerce Website

A clean, responsive, and modern **front-end e-commerce website** built using **HTML5 and CSS3**.  
ShopSphere is a multi-page electronics shopping website designed as a **college major project** with a premium UI inspired by Apple, Samsung, and Croma.

---

## ✨ Live Preview
https://shop-sphere-e-commerce.netlify.app/

---

## 📌 Project Overview

**ShopSphere** is a static e-commerce website that showcases a modern online shopping experience for electronics such as:

- 📱 Smartphones  
- 💻 Laptops  
- ⌚ Smartwatches  
- 🎧 Accessories  

It focuses on **UI/UX design, responsiveness, and professional layout structure**.

---

## 🎨 Features

✔ Modern homepage design  
✔ Responsive layout (mobile + desktop)  
✔ Clean navigation bar  
✔ Hero section with CTA  
✔ Product cards with hover effects  
✔ Category browsing sections  
✔ Special offers banner  
✔ Testimonials section  
✔ Newsletter subscription UI  
✔ Professional footer layout  
✔ Consistent color theme (Blue + Cyan tech style)
